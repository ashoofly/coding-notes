# Pipenv Quickstart

1. Create new directory for new project. Create new virtualenv with Python3 installed via Brew.  
    ```
    pipenv --python 3.8
    ```

2. Install dependency
    ```
    pipenv install pandas
    ```

3. Activate Pipenv shell
    ```
    pipenv shell
    python --version
    ```

4. Run your program

5. Deactivate Pipenv shell
    ```
    exit
    ```
